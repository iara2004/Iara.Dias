Certificados adquiridos

Ao longo da minha formação, aproveitei diversas oportunidades de aprendizado para me especializar em áreas essenciais da tecnologia, sempre buscando me destacar e me preparar para os desafios do mercado de trabalho. Os cursos online que realizei, fornecidos pelos meus professores, não apenas complementaram o conhecimento adquirido em sala de aula, mas também me proporcionaram habilidades práticas em Python, SQL, segurança cibernética, desenvolvimento web e no uso de ferramentas No-Code, áreas fundamentais para a criação de soluções inovadoras e eficazes. A seguir, apresento os certificados que conquistei, os quais refletem o meu compromisso com o crescimento contínuo e com a qualidade na entrega de resultados.

Lista de Certificados Obtidos

Cidadão Ciberseguro

Durante este curso, adquiri noções teóricas fundamentais sobre ciber-higiene e boas práticas de cibersegurança. O objetivo foi aprender a proteger a minha privacidade e segurança online, abordando aspectos cruciais como a criação de senhas fortes, a importância da atualização de software, o uso seguro de redes Wi-Fi e a prevenção de ataques de phishing e malware. Além disso, aprendi a identificar riscos e a adotar comportamentos responsáveis no ambiente digital, um conhecimento essencial para garantir uma navegação segura e proteger tanto os meus dados pessoais quanto os dos outros.

Intro to SQL

Neste curso, aprendi a gerir grandes conjuntos de dados e a analisar informações reais utilizando a linguagem padrão de gestão de dados, o SQL. Através de exemplos práticos, adquiri habilidades essenciais para a criação, consulta e manipulação de bases de dados relacionais. Com isso, tornei-me mais capaz de organizar dados de forma eficiente, realizar pesquisas avançadas e aplicar comandos como SELECT, INSERT, UPDATE e DELETE, que são fundamentais para trabalhar com bancos de dados em projetos reais.

Intro to No-Code

Neste curso de nível iniciante, aprendi a construir a minha primeira aplicação web sem necessidade de escrever código, utilizando ferramentas No-Code. O curso abordou os conceitos básicos de criação de sites e aplicativos interativos de maneira simples e intuitiva, através de interfaces visuais. Aprendi a estruturar páginas, adicionar funcionalidades e integrar diversas ferramentas, proporcionando uma experiência prática de como criar soluções digitais de forma rápida e acessível, mesmo sem conhecimentos técnicos avançados de programação.

Fundamentos de Python

Neste curso, aprendi os conceitos fundamentais de programação de computadores e comecei a desenvolver habilidades de codificação utilizando a linguagem Python. O curso cobriu tópicos essenciais como variáveis, tipos de dados, operadores, estruturas de controle (condicionais e loops) e funções. Através de exercícios práticos, fui capaz de consolidar esses conceitos e iniciar a resolução de problemas reais, preparando-me para criar programas mais complexos e eficientes. Este curso foi uma base sólida para aprofundar os meus conhecimentos em Python e me preparar para projetos de maior escala.

Launch your side project with JavaScript

Neste curso, aprendi os conceitos fundamentais do JavaScript, que me permitiram dar os primeiros passos no desenvolvimento de projetos pessoais. O curso abordou tópicos essenciais como tipos de dados, estruturas condicionais, loops e muito mais, fornecendo uma base sólida para começar a programar de forma interativa e dinâmica. Com esses conhecimentos, pude aplicar JavaScript para criar funcionalidades em websites e desenvolver projetos práticos, explorando as possibilidades dessa poderosa linguagem de programação e abrindo portas para o desenvolvimento de aplicações web mais complexas.
